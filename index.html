Primer punto

Instrucciones de usuario

Utilice el formulario que tendrá a su disposición
para enviar un mensaje.
Recuerde, es vital no dejar ningún campo vacío.
Para ayudarle, en caso de dejar un campo vacío, se mostrará una alerta
justo debajo del formulario en donde
se encuentra una lista de errores cometidos durante el llenado.
Además, una vez hecho esto, el formulario se enviará automáticamente.
Se abrirá una página de prueba, y en la página principal se podrá abrir la
consola para ver los datos impresos, además de que estos también aparecerán
en la parte de abajo del formulario.
